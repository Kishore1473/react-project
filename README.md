Event Booking System

The Event Booking System is a web application developed using React (Vite) that allows users to browse and book events through a simple and interactive interface. The project demonstrates the use of modern frontend development practices such as component-based architecture and responsive design.


---

Features

View available events

Simple event booking interface

Clean and responsive UI

Fast development using Vite

Reusable React components
